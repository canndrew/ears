Play Music easily.

Simple class to play musics easily in 2 lines.

The musics are played in them own task and load the samples progressively using circular buffers. They are not associated to a MusicData like Musics.

Examples

extern crate ears;
use ears::Music;

fn main() -> () {
   // Load a Music
  let msc = Music::new(~"path/to/my/Music.flac").unwrap();

  // Play it
  msc.play();
}
