About ears

Easy API in Rust to play Sounds.

ears is build on top of OpenAL and SndFile and is design to use sound in your Rust application in a simple way.

Features

  • 3D audio features thanks to OpenAL
  • Lot of supported audio format thanks to SndFile
  • Playing sounds ( short audio sample stored in memory )
  • Playing musics ( longer audio sample )
  • Record sounds
  • KISS: just to LOC to play a sound
